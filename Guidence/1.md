# QEMU 编译与启动

## ① 安装依赖

```bash
# 构建工具链
sudo apt install python3-venv ninja-build

# 核心开发库
sudo apt install libglib2.0-dev libpixman-1-dev libslirp-dev libfdt-dev

# GTK 图形界面支持
sudo apt install libgtk-3-dev libvte-2.91-dev

# 运行时音频库（编译不报错，但运行时缺它会报 cannot open shared object）
sudo apt install libsndio7.0
```

## ② 配置（configure）

```bash
cd qemu-10.2.4
./configure \
    --prefix="$PWD/../output/qemu" \
    --target-list=riscv64-softmmu \
    --enable-gtk \
    --enable-virtfs \
    --disable-gio
```

配置成功的标志文件：`build/config-host.mak`

## ③ 编译

```bash
make -j$(nproc)
```

## ④ 安装

```bash
make install
```

将编译产物安装到 `--prefix` 指定的目录，最终二进制位于：

```
output/qemu/bin/qemu-system-riscv64
```

## ⑤ 启动

```bash
./output/qemu/bin/qemu-system-riscv64 \
    -M quard-star \
    -m 1G \
    -smp 8 \
    -nographic --parallel none
```

## 一键构建

项目根目录的 `build.sh` 封装了 ②③④ 全流程：

```bash
./build.sh
```

脚本通过检测 `build/config-host.mak` 是否存在来判断是否需要重新 configure，避免重复配置。
